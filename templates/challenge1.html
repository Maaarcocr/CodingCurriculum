In python we can also print numbers. For example:

print 10

In this example the print command is followed by the number 10. Notice how we did not use quotation marks around the number 10, this is because we want to print the number 10. The above example will print:

10

We can also print the output of a calculation, we will call these calculations expressions. For example:

print 10 * 5
print 5 + 20 - 6
print (30 / 5) * (2 + 7)

In this example, we have several print command followed by expressions (calculations). Python will figure out the result of an expression like 10 * 5 and output it using the print statement. The code above will print:

50
19
54

It is important to remember that we didn't put the numbers or expressions in quotes, as quotes turn.

Variables!

Now that we have learnt to print content to the console, we can think about how to store information.A variable is like a container that can hold different values. We can declare a variable and insert something into it like below.

detective = "Detective Maria"

The string "Detective Maria" is stored inside the variable detective. Let's look at another example:

number_one = 22
number_two = 33
print number_one + number_two

In this example we declare two variables, number_one and number_two. Inside them we store the two values 22 and 33, respectively. With the print statement the expression number_one + number_two evaluates to 22 + 33. The result of this is then output.

55

We can also store multiple values inside a variable, this type of variable is called a list. Let's look at the example below.

numbers = [ 2, 5, 3, 6 , 8 ]

Here we have declared a variable called numbers and inside it we have used list notation [ ] to store 5 different integers. After each item in the list, we add a comma. This tells the list that we are adding a new item. If we print the list:

print numbers

We get the output:

[2, 5, 3, 6 , 8]

If we want to access a specific item inside the list, we can use list bracket notation to do this. We use the name of list variable and the order in which the item appears in the list, this is called the list index. For example:

print numbers[3]

Will output:

6

Notice in the brackets we put the number 3, but instead the 4th number was output, which was 6. This is because list indices don't start at 1 instead they start at 0. By doing numbers[3] we are accessing the 4th item in the list.
