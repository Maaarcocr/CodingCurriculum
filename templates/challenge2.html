Conditional Statements

Welldone!!! You nailed it. Now it’s time to learn something magical. You should be able to tell us what do you want us to do if a condition happens, for example if the TeamLag attacked us or stole some private information. You need to tell us what to do if a condition is true/false. That’s where Conditional Statements come in handy. Before introducing the Conditional Statements you need to know what are boolean expressions. A boolean expression is a statement which is either True or False. You can use below expressions to create a boolean statement:

> greater
< smaller
>= greater or equal
<= smaller or equal
== equal
!= not equal

In each Conditional Statement you have to use a boolean statement as the condition. The first Conditional Statement which you need to learn is “If statement”, when you want to check whether a statement is true or false you have to write: 

if condition:

And then you have to write the statement you want us to do if the condition holds:

if condition:
    statement1
    statement2
    …

The second Conditional Statement is if/else, which comes in handy when you want us to something else when the if condition doesn’t hold:

if condition:

    statement1

    statement2

    …

else:

    statement1

    statement2

    …

The third Conditional Statement is if/elif/else, you can use that when you have more than one condition that you want to check individually. You can have as many elifs as you need:

if condition:

    statement1

    statement2

    …

elif condition:

    statement1

    statement2

    …

else:

    statement1

    statement2

    …

Here are some example which might help you get along with easier:

command = “”


if command == “Attack the enemies” :

    return “Attack”

elif command == “Change The Secret Password”:

    return “ChangePassword”

elif  command == “Set an Immediate Meeting”:

    return “SetMeeting”

else

    return “EnterValidCommand”
