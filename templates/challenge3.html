For Loops

When we write programs we often find that we have to repeat a bit of code over and over and over again as you have seen previously. This is where loops come in handy! Loops can often save a lot of time writing trivial code over and over again. I will be teaching you For loops in this lesson and for loops are very similar to what we covered in while loops as they both cover the same idea.

The for loops was created since we realised we would want to do the same three things over again which are: creating loop counter variables, incrementing the variables by some amount and checking them that they are less than a certain value. This is why the for loops holds a place for all those three things and makes it very easy to work with!

A structure of a for loop in Python:

for variable in sequence:
    statements
else:
    statements

A simple example of a for loop:

languages = ["Spanish", "English", "Persian", "Arabic"]
for x in languages:
    print

The result of that program will be:

Spanish
English
Persian
Arabic

Another example is using the range function. The range function uses two arguments like thisRange(start, finish). start is the first number that is produced. finish is one larger than the last number. For instance:

oneToSix = range(1, 7)

for count in oneToSix:
    print count

The output for this is:

1
2
3
4
5
6

Furthermore, you can also find a position of a character in a string or list/arrays using for loops. To do this you usually will need to create a variable counter which will be printed whenever the expression is met. Here’s how it works:

Input

myString = "Position of a character"

count = 0
for c in myString:
    if "r" == c: print count
    count = count + 1

Output

17
22

As you can see the letter r is in positions 17 and 22, don’t forget when we’re counting we always start from 0!
